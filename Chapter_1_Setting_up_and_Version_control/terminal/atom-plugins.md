All the following atom packages can be installed from your terminal. Just copy every line without the `$`

### Javascript

```
$ apm install atom-ternjs
$ apm install atom-typescript
```

### Testing

```
$ apm install cucumber
$ apm install cucumber-autocomplete
$ apm install cucumber-step
$ apm install gherkin-autocomplete

$ apm install rails-rspec
$ apm install rspec
```

